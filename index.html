HTML & CSS Basics

Frequently asked questions about HTML and CSS.

  • What is HTML?

    + -

    HTML stands for HyperText Markup Language. It is used to create the structure of web pages by defining elements such as headings, paragraphs, links, images, and more.

  • What is the purpose of CSS?

    + -

    CSS, or Cascading Style Sheets, is used to control the appearance and layout of web pages. It allows developers to style elements, such as setting colors, fonts, spacing, and responsive designs.

  • Class vs. ID?

    + -

    Classes are used to style multiple elements with the same properties, while IDs are unique and should be used for only one element on a page. In CSS, a class is selected with a dot (`.`) and an ID with a hash (`#`).

  • What is boxing in CSS?

    + -

    The CSS box model describes the rectangular boxes generated for elements. It consists of margins, borders, padding, and the content area, all of which determine the size and spacing of elements.

  • Responsive Design?

    + -

    Responsive design adjusts layouts using media queries to fit different screen sizes.
